# 🍔 MenuYa App - Delivery Service

¡Bienvenido a **MenuYa**! Esta es una aplicación móvil desarrollada con **Flutter** diseñada para ofrecer una experiencia de usuario fluida y moderna en la gestión de pedidos de comida, inspirada en plataformas líderes de delivery.

---

# 🚀 Características Principales

### 🍕 Exploración de Productos
Catálogo dinámico de platillos con imágenes y descripciones detalladas.

### 🛒 Gestión de Carrito
Agregado y edición de productos en tiempo real antes de finalizar la compra.

### ⭐️ Sistema de Favoritos
Guarda tus platos preferidos para un acceso rápido.

### 👤 Perfil de Usuario
Gestión de datos personales y preferencias.

### 💳 Pasarela de Pago
Simulación de proceso de pago integrada.

### 📱 Interfaz Adaptable
Diseño responsive que funciona tanto en Android como en iOS.

---

# 🛠️ Tecnologías Utilizadas

* **Frontend:** Flutter & Dart
* **Gestión de Estado:** StatefulWidgets
* **Iconos y Estilo:** Material Design 3
